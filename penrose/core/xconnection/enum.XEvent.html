Wrapper around the low level X event types that correspond to request / response data when communicating with the X server itself.

The variant names and data have developed with the reference xcb implementation in mind but should be applicable for all back ends.

Variants

MouseEvent(MouseEvent)

The mouse has moved or a mouse button has been pressed

KeyPress(KeyCode)

A grabbed key combination has been entered by the user

MapRequest

A client window is requesting to be positioned and rendered on the screen

rpt: Point

wpt: Point
