[−][src]Module penrose::xcb

Helpers and utilities for using XCB as a back end for penrose

This is the reference implementation of the core Penrose traits that are required to back the WindowManager when talking to the X server. The code in this module is build on top of the xcb, pango and cairo C libraries and strives to be as simple as possible (as opposed to optimising for performance).

Available features

  • xcb_draw: adds pango and cairo as build dependencies and provides implementations of the Draw and DrawContext traits.
  • xcb_keysyms: if enabled, this will include the XKeySym enum which is based on the contents of X11/keysymdef.h and allows for parsing X keycode events as their corresponding utf-8 representation or action.
